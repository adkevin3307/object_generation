# Object Generation

Object generation for iCLEVR dataset w/ GAN and Normalizing Flow (GLOW)

## Environment

- `Python >= 3.9`
- `torch >= 1.9.0`
- `torchvision` w/ appropriate version

## Usage

- `python3 gan.py --help` for more details
- `python3 glow.py --help` for more details
